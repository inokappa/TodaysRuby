### Q9. オブジェクト指向

以下のコードの実行結果から \_\_(1)\_\_ に当てはまるものを選択する.

```ruby
# コード
class Foo
  def foo
   puts "foo"
  end
end
__(1)__


# 実行結果
foo
```

以下が \_\_(1)\_\_ に該当する.

* Foo.new.foo

以下, 解説より抜粋.

* クラス内で定義したメソッドはインスタンスメソッドとなる
* インスタンスメソッドは `オブジェクト.メソッド` のフォーマットで呼び出すことが出来る
* オブジェクトは `クラス.new` で生成する為, `クラス.new.メソッド` として連続して呼び出すことが出来る
